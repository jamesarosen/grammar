pronoun_or_noun(noun, audience, options = {})

Return a pronoun or a noun, depending on the relationship of noun to audience. If they are the same, infers second person and uses ‘you’. If they differ, uses noun.to_s

Options:

  • :force_pronoun - use a pronoun regardless of the relationship of noun to audience, but still infer person (so return ‘she’, ‘his’, ‘themsleves’, etc.); defaults to false
  • :force_noun - use noun.to_s, regardless of the relationship of noun to audience
  • :capitalize - capitalize the result, but only if it‘s a pronoun; defaults to false
  • :case - the case of the pronoun or noun; defaults to :subject
  • :gender - the gender of the pronoun to use (used only if :force_pronoun in English; other languages have gendered second-person pronouns).
