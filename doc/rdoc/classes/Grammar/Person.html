parse(i, j = nil)

Converts i and j to one of [FIRST, SECOND, THIRD] as follows:

  • if i is a Symbol, i
  • if i is one of [1, 2, 3], returns FIRST, SECOND, or THIRD, respectively
  • else treats i and j as subject and addressee, respecively, returning SECOND if i and j are equal, otherwise THIRD.

Raises ArgumentError if the result is not a valid verb person.
