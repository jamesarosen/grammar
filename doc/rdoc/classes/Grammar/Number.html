parse(n_or_subject)

Converts n_or_subject to :singular or :plural as follows:

  • if n_or_subject is Nil, :plural
  • if n_or_subject is a Symbol, do nothing to it
  • if n_or_subject is a Numeric, :singular if n_or_subject == 1, else :plural
  • if n_or_subject responds to many?, :plural if n_or_subject.many?, else :plural
  • if n_or_subject responds to several?, :plural if n_or_subject.many?, else :plural
  • if n_or_subject responds to size (but is not a Numeric), convert to a Numeric, then process as above
  • if n_or_subject responds to length (but is not a Numeric), convert to a Numeric, then process as above
  • otherwise, :singular
